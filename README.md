# 🍔 FoodSocial – Social Food Ordering Platform

FoodSocial is a full-stack MERN application that combines the experience of **Instagram reels** with **Zomato-style food ordering**.

Users can explore food reels, like, comment, save posts, and order food directly from posts.

---

## ✨ Features

### 👤 Authentication
- User & Food Owner roles
- Login / Signup with JWT authentication

### 🎬 Social Feed
- Upload food reels/posts
- Like, Comment, Save posts
- Personalized **Mood-Based Feed**

### 😊 Mood Based Feed
Filter posts based on user mood:
- Happy
- Lazy
- Healthy
- Budget
- Party

Mood preference is saved using localStorage.

### 🛒 Order From Reel
- Food owners can mark posts as orderable
- Users can order food directly from reels
- Cart drawer with checkout UI

### 🗺️ Additional Features
- Visit Store option
- Persistent user experience
- Responsive modern UI
